# Coding the Car

* Set radio group (**same as sender**) in the on start code block
* Put all your logic statements in an on radio received [value] code block.
<br>

Note: The below radio inputs may be different, refer to your controller code.
* If "F" is received move forward at 100 speed
* If "L" is received move left at 100
* If "R" is received move right at 100
* If "B" is received reverse at 50 speed
* If "stop" is received then stop.

* Feel free to update the values you want. Trial and error is key!
